2. Promedio de las variables cuantitativas

Antes de calcular los promedios, es importante destacar que este an√°lisis se enfocar√° en las variables cuantitativas del conjunto de datos, espec√≠ficamente: calor√≠as (calories), prote√≠na (protein), sodio (sodium), az√∫car (sugars) y potasio (potass). Estas variables permiten realizar operaciones aritm√©ticas, como el c√°lculo de promedios, debido a su naturaleza cuantificable, a diferencia de las variables cualitativas, que no son susceptibles a tales c√°lculos.

# 1. Selecci√≥n de variables cuantitativas y c√°lculo de promedios
tabla_promedios <- cereal1 %>%
  summarise(
    `Calor√≠as (calories)` = mean(calories, na.rm = TRUE),
    `Prote√≠na (protein)` = mean(protein, na.rm = TRUE),
    `Sodio (sodium)` = mean(sodium, na.rm = TRUE),
    `Az√∫car (sugars)` = mean(sugars, na.rm = TRUE),
    `Potasio (potass)` = mean(potass, na.rm = TRUE)
  ) %>%
  pivot_longer(cols = everything(),
               names_to = "Variable",
               values_to = "Promedio") %>%
  mutate(Promedio = round(Promedio, 2))

3. Proporci√≥n de cereales fabricados por fabricante

Aunque el fabricante es una variable cualitativa, es posible calcular proporciones basadas en sus frecuencias absolutas. Estas proporciones indican la contribuci√≥n de cada fabricante al total de cereales analizados, proporcionando una visi√≥n clara de la distribuci√≥n de productos por empresa en el conjunto de datos.

# 1. Vector para cambiar las abreviaturas de los fabricantes por los nombres originales 
fabricantes <- c("A" = "American Home Food Products",
                 "G" = "General Mills",
                 "K" = "Kellogg's",
                 "N" = "Nabisco",
                 "P" = "Post",
                 "Q" = "Quaker Oats",
                 "R" = "Ralston Purina")

# 2. C√°lculos para la creaci√≥n de la tabla de frecuencias y proporciones
tabla_fabricantes <- cereal1 %>%
  count(mfr) %>%
  mutate(
    Fabricante = fabricantes[mfr],  # Reemplazar abreviaturas por nombres completos
    `Frecuencia Absoluta` = n,
    `Proporci√≥n` = round(n / sum(n), 4)
  ) %>%
  select(Fabricante, `Frecuencia Absoluta`, `Proporci√≥n`) %>%
  arrange(desc(`Frecuencia Absoluta`))
